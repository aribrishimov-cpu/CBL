# Cook or Get Cooked

**Project:** Cook or Get Cooked  
**Date:** 05.10.2025   
**Team:** Atanas, Lucas  

## Overview
Cook or Get Cooked is a 2D restaurant management game. Collect ingredients, cook or chop them, assemble burgers, and deliver orders before the timer runs out.

##Controls
Arrow keys → Move player

E → Pick up ingredient

Q → Drop / stack ingredients

Space → Chop ingredient

Check the terminal for messages when playing.
